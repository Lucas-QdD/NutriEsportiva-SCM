# SweatApp

Aplicativo para avaliação da taxa de sudorese e suporte à tomada de decisão em hidratação de atletas.
README inicial e criado apenas para contexto.

## Tecnologias
- React Native + Expo
- Next.js
- Node.js + Express
- Prisma
- SQLite

## Estrutura
- `mobile/` aplicativo do atleta
- `web/` painel do treinador/nutricionista
- `backend/` API compartilhada
- `docs/` documentação do projeto
